# CareerTrackAI

CareerTrackAI is an AI-powered career assistant built with Streamlit. It helps users analyze resumes, identify skill gaps, generate ATS-friendly resume suggestions, cover letters, and career guidance. It also provides job recommendations based on user skills and preferences.

## Features

- **Resume Analysis**: Upload a resume (PDF or TXT) and get an ATS score and improvement suggestions.
- **Skill Gap Analysis**: Identify missing skills and learning opportunities.
- **Job Recommendations**: Find jobs matching your skills, location, experience, and job type.
- **Cover Letter Generation**: Generate professional cover letters tailored to your resume.
- **Career Advice**: Get optimal career path suggestions.
- **Quick Action Buttons** for fast analysis and recommendations.

## Tech Stack

- Python
- Streamlit
- LangChain (LLM integration)
- FAISS for embeddings
- PyPDF for PDF parsing
- dotenv for environment variables
- BeautifulSoup, lxml for web scraping
